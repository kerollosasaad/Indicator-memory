Usage
-----

```
sudo apt-get install build-essential libgtop2-dev libgtk-3-dev libappindicator3-dev git-core
git clone https://github.com/kerollosasaad/Indicator-memory
cd Indicator-memory
make
sudo make install
Indicator-memory &
```
